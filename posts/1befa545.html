使用 git + Markdown 做笔记

git 对程序员一点都不陌生，而 Markdown 更不用说了。

但是对非程序员的其他人来说，这东西能干嘛都不知道，更别说用来写笔记。

git

git 是 Linux 作者 Linus 自己开发的代码版本管理工具。用于回溯代码，只要通过 git 提交过的版本，都能通过 git 找回来。

Markdown

Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档。

说人话就是，你只需要关系你写的东西，而不需要在写东西的时候就过度关注你的样式。

直到你写完后，通过不同的解析器导出带有格式的文件。

Start

需要先安装 git bash 和 visual studio code

安装好 git bash 后，在你的笔记所在的目录中右键-> git bash，会弹起一个黑窗口，在黑窗口中输入下面的的命令打开 visual studio code。

Init

如果你第一次在文件夹中使用 git，需要在目录中执行 init 命令，创建一个 git 本地仓库。

使用 git 保存你的笔记

当你编辑完笔记（或者你需要中断写作），最好使用 git 保存下笔记。
