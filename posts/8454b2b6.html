工具

image-to-ascii

把图片文件生成为 ascii。

maza-ad-blocking

一个类似 Pi-hole 的广告屏蔽工具

onefetch

使用方式和neofetch很像的一个命令行工具，用于展示 git 管理的项目的汇总信息。

git-stats

本地 git 统计数据，包括类似 GitHub 的贡献日历

github-readme-stats

github 统计卡片

git-quick-stats

访问 git 存储库中各种统计信息的简单有效的工具。

yapi

在多人协作开发的时候，文档是很重要的，没有文档的开发任务，前端和后端都没办法开工。YAPI 提供一个 api 管理的服务解决这个问题。
