工具

YYeTsBot

🎬 人人影视 bot，完全对接人人影视全部无**资源。

pi-ap

树莓派做为 AP 的脚本工具，哪天应急说不定就用上了。

mycli

mysql 命令行工具，有高亮代码提示，比 mysql-client 好用。

ali

咋一看，我以为是阿里巴巴又出了什么新玩具，其实不是，这是一个用 go 实现的 http 压测工具。

httpstat

类似于 curl -v, 有颜色。

vscode-qq

基于安卓 QQ 协议的 vscode-qq 扩展，万能的 vscode, 以后也能在 linux 上愉快的聊 qq 了。

swdc-vscode-musictime

Music Time for Spotify, 一边听音乐一边码字。不过我还是喜欢用网易云音乐。

openwrt

开源路由系统，想必玩过软路由的都知道这个东西。

tree

在 linux 下有个 tree 命令，可以以树状图的形式在命令行打印文件夹的结构。这个项目用各种语言实现了这个命令。

youtube-dl

命令行下载 youtube 视频的工具，之前还被强制删 github 仓库，不过现在又可以用了。

czkawka

之前一直在找对比差异文件的工具，这个就很不错，而且是 rust 写的

资源

《JavaScript 二十年》 中文版

本书由 JavaScript 之父 Brendan Eich 与 ES6 规范首席作者 Allen Wirfs-Brock 联合编写，详细记载和解读了自 1995 年语言诞生到 2015 年 ES6 规范制定为止，共计 20 年的 JavaScript 语言演化历程。

ES2021 新特性

本文介绍了 JavaScript 今年新增了哪些语法，还好不是很多。
