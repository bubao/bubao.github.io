周刊 （第 1 期）

一直以来都是看别人写的周刊和月刊，然后 star 到自己的 github 仓库就忘记了。不如自己也整一个，把自己每一周看到的好项目以及学习资料收集起来，方便自己查阅。

工具

keppler

通过浏览器，实时代码共享，可以用来在局域网内授课和 code review。

easings.net

常用缓动函数库，帮助开发者选择正确的缓动功能的备忘录。

SvgPathEditor

在浏览器上常见和编辑 svg path。

mitojs

一款轻量级的收集页面的用户点击行为、路由跳转、接口报错、代码报错、并上报服务端的 SDK

svgi

命令行审查 svg 结构的工具。

piscina

Node.js 线程池

suncalc

一个可以计算日出日落的 Nodejs 库

1loc

这个网站上收集了一行 js 实用代码。

better-sqlite3

Node.js 操作 sqlite3 的库，notion 的 github 账号 fork 过。

一直以为 notion 官方没有 github 账号，通过这个包意外发现 github.com/makenotion，另外是不是意味着 notion 打算做本地笔记了？

notion-sdk-js

notion 官方 JavaScript Notion api。
